Redminetimetracker

Chrome app for tracking work time.

Time Tracking tool for Redmine.

Redmineの作業時間を記録するためのツールです. チケット毎の工数管理や生産性の把握に使用することができます.

Chrome Web Store

使い方

0. RedmineのREST APIの有効化

本ツールはRedmineのREST APIを使用します. REST APIが有効になっていない場合, 以下のページを参考にREST APIを有効にしてください. 既に有効になっている場合, 本手順は不要です.

Redmine REST API
http://www.r-labs.org/projects/r-labs/wiki/Redmine_REST_API#ユーザー認証

REST APIの有効化にはRedmineのシステム管理者権限が必要です. システム管理者権限をお持ちでない場合は, システム管理者にREST APIの有効化を依頼してください.

1. Redmineの登録

RedmineのURLと認証情報を登録します. 認証情報には (1)UserIDとパスワード, 又は (2) ApiKey が使用できます. 使用する認証情報に従って(1)又は(2)のいづれかを行ってください.

(1) UserIDとパスワード Redmineのログインに使用するUserIDとパスワードを登録します. 画面をスクロールし, 'Settings' を表示します。 URLとUserID, パスワードを入力してください.

(2) ApiKey Redmineのログインに使用するApiKeyを登録します. 画面をスクロールし, 'Settings' を表示します. URLとApiKeyを入力してください. なお、ApiKeyは'個人設定'ページの右側に表示されます.

これらの認証情報は暗号化された上でGoogle Chrome上で同期されます. そのため, 一度登録するとログインしている全てのGoogle Chrome上で 本ツールの情報が同期されます.

2. チケットの選択

Redmineの認証に成功すると'Time Tracking'及び'Edit List'に チケットが表示される様になります. 認証直後は'Time Tracking'には自分の担当チケットのみが表示されます. 担当外のチケットを選択する場合は'Edit List'から選択してください.

3. 作業時間の記録

'Time Tracking'でチケットを選択し, 開始/停止ボタンで 作業時間の記録を開始します. もう一度ボタンを押すとRedmine上に作業時間が記録されます.
